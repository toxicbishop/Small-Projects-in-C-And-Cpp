## ğŸ› ï¸ How to Compile and Run

### Prerequisites

- GCC compiler (MinGW on Windows, or any C compiler) for C projects.
- G++ or equivalent for future C++ projects.

### Compilation

```bash
# Navigate to the project folder
cd C/Casual_Projects/Project-Folder-Name

# Compile the program
gcc program-name.c -o program-name

# Example: Compile the number guessing game
cd C/Casual_Projects/Number-Guessing-Game
gcc Num-guessing-game.c -o Num-guessing-game
```

### Running

```bash
# Run the compiled executable
./program-name

# On Windows
program-name.exe
```
